# 📜 POLÍTICA DE PRIVACIDAD CertiDocs

**Última actualización: 19/04/2026**

---

## 1. DATOS QUE RECOGEMOS
✅ Correo electrónico
✅ DNI / NIF (únicamente para tramitación de certificados oficiales)
✅ Datos de la solicitud que introduce el usuario
✅ Token de dispositivo para notificaciones push
✅ No recogemos datos de localización
✅ No compartimos ningún dato con terceros

## 2. FINALIDAD DEL TRATAMIENTO
Todos los datos recogidos son exclusivamente para:
- Tramitar los certificados oficiales ante el Ministerio de Justicia
- Enviar notificaciones sobre el estado de las solicitudes
- Gestión de la cuenta de usuario

## 3. DERECHOS DEL USUARIO
Puede solicitar en cualquier momento:
✅ Acceso a sus datos
✅ Rectificación de datos incorrectos
✅ Eliminación completa de su cuenta y todos sus datos
✅ Oposición al tratamiento

Contacto: `privacidad@certidocs.es`

## 4. SEGURIDAD
- Todas las comunicaciones son cifradas HTTPS
- Las contraseñas se almacenan con hash bcrypt
- No almacenamos datos de tarjetas de crédito (Stripe gestiona íntegramente los pagos)

## 5. MENORES DE EDAD
Esta aplicación no está destinada a menores de 14 años.
